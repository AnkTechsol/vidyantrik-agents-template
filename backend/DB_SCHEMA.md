## Collections

### 1. Users
```javascript
{
  name: String (required),
  email: String (required, unique, indexed),
  password: String (required, hashed),
  role: String (enum: ['student', 'instructor', 'admin'], default: 'student'),
  enrolledCourses: [ObjectId] (ref: 'Course'),
  createdAt: Date,
  updatedAt: Date
}
```

### 2. Courses
```javascript
{
  title: String (required),
  description: String (required),
  instructor: String (required),
  duration: Number (hours),
  level: String (enum: ['Beginner', 'Intermediate', 'Advanced']),
  thumbnail: String (URL),
  category: String,
  modules: [{
    title: String,
    content: String,
    videoUrl: String,
    duration: Number
  }],
  enrolledStudents: Number (default: 0),
  rating: Number (default: 0),
  createdAt: Date,
  updatedAt: Date
}
```

### 3. Quizzes
```javascript
{
  courseId: ObjectId (ref: 'Course', required),
  title: String,
  questions: [{
    question: String,
    options: [String],
    correctAnswer: Number (index)
  }],
  passingScore: Number (default: 70),
  createdAt: Date,
  updatedAt: Date
}
```

### 4. QuizSubmissions
```javascript
{
  userId: ObjectId (ref: 'User', required),
  quizId: ObjectId (ref: 'Quiz', required),
  courseId: ObjectId (ref: 'Course'),
  answers: [{
    questionId: ObjectId,
    selectedAnswer: Number
  }],
  score: Number,
  passed: Boolean,
  submittedAt: Date
}
```

### 5. Certificates
```javascript
{
  userId: ObjectId (ref: 'User', required, indexed),
  courseId: ObjectId (ref: 'Course', required),
  certificateId: String (unique, auto-generated),
  studentName: String,
  courseName: String,
  completionDate: Date,
  issuedDate: Date (default: Date.now),
  verificationUrl: String
}
```

## Indexes
- Users: email (unique)
- Certificates: userId, certificateId (unique)
- QuizSubmissions: userId, quizId
